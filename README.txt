📘 國字描紅練習系統（grade3_top1_words_full_package）

------------------------------------------------------------
一、檔案內容
------------------------------------------------------------
此套件包含三個主要檔案：

1. index.html
   └ 主頁面，包含「設定」與「練習」兩個分頁介面。
2. app.js
   └ 主程式，負責描紅邏輯、出題規則、權重比例與統計功能。
3. data_g1_tog31.js
   └ 整合資料庫，包含「小一下」、「小二上」、「小二下」、「小三上」四學期共 48 課的字與注音。


------------------------------------------------------------
二、使用方式
------------------------------------------------------------
1. 將三個檔案放在同一資料夾中（例如 /grade3_top1_words/）。
2. 直接開啟 index.html 即可使用（支援桌機與手機瀏覽器）。
3. 若要上傳到 GitHub Pages，放在 repo 根目錄並啟用 Pages 功能即可。

初次開啟預設為「設定」分頁，點選上方「練習」即可開始描紅。


------------------------------------------------------------

------------------------------------------------------------
