# 國字描紅 PWA 套件（含 index.html）

把以下檔案整包複製到你的 GitHub Pages 專案根目錄（grade3-top1-words/）即可：
- index.html（已內建 PWA tags 與 SW 註冊）
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png
- icons/icon-512-maskable.png

> 注意：這個壓縮檔不包含 app.js / data.js，請保留你原本的檔案在同一層。

完成後用 iPhone Safari 開啟網站，分享 -> 加入主畫面。
